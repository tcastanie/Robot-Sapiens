Installation Type Selection

Full

Installs all Madkit plugins, including

  • Desktop 2 and a default configuration
  • All sources
  • Documentation (User guide, Developper Guide and API documentation)

Custom

Select this option if you don't need all the plugins. You can select the plugins to install inside the Plugin Manager

Install Source

Lets you select the source to use for the install.

Internet: If you selected Internet as your Install Source in the previous step, You can enter any http repository. Normally http://www.madkit.net/plugins is fine.

Local File: If you selected Local File as your Install Source in the previous step, You can select the madkit bundle file. Click on Browse to select the file.

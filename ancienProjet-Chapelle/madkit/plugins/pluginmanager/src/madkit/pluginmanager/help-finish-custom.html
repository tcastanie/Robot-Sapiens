Custom Install

You can selected a Custom Install.

Check if all the Information is Correct, if not, use the Back button to correct it.

The Installation Process in not complete yet. Click on Launch Plugin Manager to select the plugins you want to install.

When you are done, click on Install in the Plugin Manager Window, this will start the installation process.

Which plugins should I select?

The only plugin that you Must Install is madkit-kernel plugin. This provides the Madkit Kernel and main classes (like Agent, AbstractAgent, Message, etc).

You can get a small description of each plugin by selecting the plugin and then clicking on Details in the plugin manager window. The information includes dependencies, size of the plugin and a description of the plugin purpose

Here is a small list of what you might need:

Desktop
If you want to install Madkit desktop you need to select desktop2. You might want to install desktop2-commons to get a default configuration
Communication
If you want to connect several kernels, you need the netcomm or the communicator plugin
Turtlekit
Look in the Turtlekit Category
Documentation
You can install all the documentation by selecting madkit-doc bundle. You can also select them independently, each plugin provides its own documentation bundle.
Source
You can install all sources by selecting madkit-src bundle. You can also select them independently, each plugin provides its own source bundle.

Important: Even if you selected a custom install, you can always make a full install inside the Plugin Manager by selecting madkit-full.
Don't worry about the dependencies, the Plugin Manager will add to the list any plugin needed by the plugins to be installed.

1. Summary

This agent pack gathers some classic agent demonstration and shows some functionalities of the MadKit kernel and its associated libraries.

This pack includes the following agents:

  • PingPong. A simple message, and group & role example

  • LiveScheme. An agent embedding the Kawa Scheme implementation, enabling dynamic agent programming.

  • Minimal. Basic agent lifecycle example

  • SimplePlotAgent. An example of an agent which defines its own GUI

  • BeeScheduler / BeeWorldViewer. Two simple agents demonstrating the synchronous engine and the probes / activators mechanism.
